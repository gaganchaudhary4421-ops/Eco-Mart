# Eco-Mart — E-Commerce Frontend

## Tech Stack

- React.js
- Bootstrap 5
- CSS3
- Bootstrap Icons

## Sections

- Announcement Bar
- Header with Search
- Hero Banner with Category Sidebar
- Flash Sales with Countdown Timer
- Browse By Category
- Best Selling Products
- JBL Music Banner
- Explore Our Products
- New Arrival
- Service Features
- Footer

## Getting Started

### Prerequisites

- Node.js v16+
- npm

### Installation

```bash
git clone https://github.com/gaganchaudhary4421-ops/Eco-Mart.git
cd eco-mart
npm install
npm start
```
